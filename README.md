#  Linux Market Monitoring Tool

## 🔍 Overview

Personal project exploring Linux system workflows by building a market monitoring tool within an Ubuntu environment.

The project retrieves financial market data, analyses daily price movements, and generates automated reports while developing practical experience with Linux tooling and command-line workflows.

## ⚙️ Features

- Retrieves and processes market data
- Calculates daily price movements
- Generates monitoring reports
- Logs execution output and results
- Uses Linux-based scripting and automation workflows
